Fly is dropdown/tooltip/popover call it whatever you want lib. It helps you to not care about positioning, showing/hiding, event racing and other boring stuff, but focus on what and when you want to show.

Fly has built-in dropdown and tooltip

The easiest way to use it is jQuery plugin: 

$('.js-dropdown')
    .dropdown({content: 'Hello'});

$('.js-tooltip')
    .tooltip({content: 'Welcome'});

You can use constructors to create more advanced instances: 

$('.js-dorpdown').each(function() {
    $.fly.dropdown.create(this, { content: $(this).data('content') });
});

The whole list of options you can pass to create or jQuery.fn.dropdown: 

{
    baseClass: 'fly-dropdown',          // main root class
    hideClass: 'fly-dropdown_hidden',   // class to be added on .hide()
    extraClass: '...',                  // any additional root classes
    content: '',                        // defines how to get content
    position: 'bottom center',          // dropdown-tail position
    arrowSize: 10,                      // offset between dropdown and handle
    redrawOnShow: true,                 // recalculate content on every show event
    register$: 'titletip'               // will register titletip as jQuery plugin
}

Try to play with tooltip position: 

$('js-tooltip').tooltip({position: '', content: 'Hello'});

Content in the simplest case is just a string:

$('.js-dropdown').dropdown({content: 'Hello'});

Also you can use function to calculate some value:

$('.js-dropdown').dropdown({
    content: function() {
        return new Date();
    }
});

Fly could wait for async content resolution, if you will define first argument for the content function: 

$('.js-dropdown').dropdown({
    content: function(done) {
        setTimeout(function() {
            done('Thanks for you patience');
        }, 500);
    }
});

To create your own fly class, use extend, where you can overwrite anything you want or add new: 

var myDropdown = $.fly.dropdown.extend({
    defaults: {
        content: function() {
            return this.getTime();
        }
    },
    getTime: function() {
        return new Date();
    }
});
myDropdown.create('.js-dropdown');

Fly class can be delegated, it will create instance only after user's first interaction with the handle 

$.fly.dropdown.delegate('.js-dropdown', {content: 'Hello'});

While you defining class you can register it as a jQuery plugin: 

$.fly.dropdown.extend({
    defaults: { content: 'Hello' },
    register$: 'myDropdown'
});
$('.js-dropdown').myDropdown();

You can get fly instance by handle 

var inst = $('.js-dropdown').myDropdown('instance');

You can destroy instance by handle 

$('.js-dropdown').myDropdown('destroy');
